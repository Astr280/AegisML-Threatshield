# 🚀 Quick Start Guide

## Malware Analysis and Detection System

### Prerequisites Check

Before starting, ensure you have:
- ✅ Windows 10 or 11
- ✅ Python 3.10.9 or higher installed
- ✅ Minimum 8GB RAM
- ✅ Internet connection (for initial setup)

### Installation Steps

#### Step 1: Install Python (if not already installed)

1. Download Python 3.10.9 from: https://www.python.org/downloads/
2. **IMPORTANT**: During installation, check the box "Add Python to PATH"
3. Verify installation by opening Command Prompt and typing:
   ```
   python --version
   ```

#### Step 2: Run Setup

Double-click `setup.bat` or run in Command Prompt:
```powershell
setup.bat
```

This will:
- Install all required Python packages
- Generate the malware dataset (upload.csv)
- Prepare the application for use

#### Step 3: Start the Application

Double-click `run.bat` or run in Command Prompt:
```powershell
run.bat
```

The application will start at: **http://127.0.0.1:5000/**

### First Time Usage

1. **Open Browser**: Navigate to http://127.0.0.1:5000/

2. **Login**:
   - Click "Login" in the navigation menu
   - Username: `admin`
   - Password: `admin`

3. **Upload Dataset**:
   - Click "Upload" menu
   - Select the `upload.csv` file (generated by setup)
   - Click "Upload Dataset"

4. **Train Models**:
   - Review the dataset preview
   - Click "Click to Train & Test"
   - Wait for training to complete (2-5 minutes)

5. **Make Predictions**:
   - Select app permissions (Yes/No for each feature)
   - Choose a model (Extra Trees or Logistic Regression)
   - Click "Predict"
   - View results and download PDF report

6. **Explore Analytics**:
   - **Performance**: View detailed metrics and confusion matrices
   - **Charts**: See visual comparisons and dataset distribution

### Troubleshooting

#### Python not found
- Reinstall Python and ensure "Add to PATH" is checked
- Restart Command Prompt after installation

#### Port already in use
- Close other applications using port 5000
- Or edit `app.py` and change the port number

#### Training takes too long
- Ensure you have sufficient RAM (8GB minimum)
- Close other applications to free up memory

#### Dependencies installation fails
- Update pip: `python -m pip install --upgrade pip`
- Install packages individually if needed

### Manual Installation (Alternative)

If batch files don't work, run these commands manually:

```powershell
# Navigate to project directory
cd "d:\Cybersecurity Project"

# Install dependencies
python -m pip install -r requirements.txt

# Generate dataset
python generate_dataset.py

# Run application
python app.py
```

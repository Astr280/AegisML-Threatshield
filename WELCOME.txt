┌───────────────────────────────────────────────────────────────────────────┐
│                         🔧 TROUBLESHOOTING                                │
└───────────────────────────────────────────────────────────────────────────┘

  Problem: Python not found
  Solution: Reinstall Python with "Add to PATH" checked
           Restart Command Prompt after installation

  Problem: Dependencies installation fails
  Solution: Update pip: python -m pip install --upgrade pip
           Then retry: python -m pip install -r requirements.txt

  Problem: Training takes too long
  Solution: Ensure 8GB+ RAM available
           Close other applications
           Be patient (2-5 minutes is normal)

  Problem: Port 5000 already in use
  Solution: Close other applications using port 5000
           Or edit app.py to use different port

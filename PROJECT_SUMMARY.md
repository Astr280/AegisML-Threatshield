## 🤖 Machine Learning Implementation

### Models Implemented

#### 1. Extra Trees Classifier
- **Accuracy**: 97.23% (test)
- **Type**: Ensemble method
- **Estimators**: 100 trees
- **Best For**: Maximum accuracy

#### 2. Logistic Regression
- **Accuracy**: 93.67% (test)
- **Type**: Linear classifier
- **Iterations**: 1000
- **Best For**: Speed and interpretability

### Dataset
- **Total Samples**: 4,465
- **Features**: 23 selected permissions
- **Classes**: Malware (70%), Benign (30%)
- **Format**: CSV
